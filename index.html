							<div class="d_block d-sm_inlineBlock col_24 col-sm_24 col-md_12 col-lg_8  mx_20 my_30 my-sm_20"></div>
						</div>


						- media and value are optional
						- media is always in the middle
						`-xs-` `-xsOnly-` `-md-` `-lg-` `-xl-` `-mdToLg-` `-print-` `-iron-` etc.
						- value is always last
						`-major` `-row` `-justify` `-100`  `-38parrots`
						- all the parts of a utility are in camelCase separated by only one hyphen

						`.jc-md-between`

						`.d-xsOnly-none`

						`.gutter-sm-sm` (parts position is important, middle `sm` is media, last `sm` - is value)

						`.col-sm` не подходит, т.к. media не может быть в конце. Правильно, например `.col-sm-rest`

						## Component

						`SomeComponent` `SomeComponent__innerElement` `SomeComponent--modifier`

						### Reserved prefixes (you can use more)

						`.do-` the hook for using in scripts

						`.is-` component state modifier

						`.has-` component descendant state/presence modifier

						`.g-` global context class

  • .SomeBlock - блок (TitleCase)
  • .SomeBlock__innerBlock — элемент
  • Модификаторы:
    • .SomeBlock--modifier — модификатор (camelCase)
    • .is-* — состояние блока (is-selected)
    • .has-* — наличие потомка, или состояние (has-focus например)
  • .do-* — префикс классов для навешивания скриптов
  • Сетка .row, .col-*, .grid, .gutter-*
  • Утилиты
    • Кастомные .u-whatItDoes (.u-expandLinkArea)
    • Атомарные
      • блочные: .d-*, .justify-*, .align-*
      • поля и отступы .p-* .mx-*
      • текстовые: .text-*, .font-*, .color-*, .list-* (.font-size-sm)
